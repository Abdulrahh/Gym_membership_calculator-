# Gym_membership_calculator-
class activity 

Customer Details: Collect the Member Name and a unique Membership ID.
Validation:
Membership ID must be exactly 6 characters long and must end with "-M" (e.g., "1234-M").
Age must be a positive integer between 16 and 90.
----
Selection (Logic):
Bronze Tier: £25.00 per month.
Silver Tier: £40.00 per month.
Gold Tier: £60.00 per month.

Student Discount: Students receive a 20% discount on the monthly fee.
Output Summary: An itemized "Membership Summary" showing Member Name, ID,
Tier, Discount applied, and Total Monthly Cost.
